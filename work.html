How many students attended per week?

This is the code to figure out the number of unique check-ins per week for any given quarter.

    def attendanceByWeek(df):
        dfAttendanceWeek = df.assign(Count = df.get('Timestamp')).groupby('Week').count()
        return dfAttendanceWeek.get(['Count'])
